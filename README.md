## Description

**Plain Plane Tracker** is a desktop application that allows users to search, monitor. and visualise real-time flight data. The app provides semi-live updates on flights by airline, flight numbers, or airport (destination or arrival), and displays their information (like altitude, and speed) in a Java Swing Interface.
